# Rogvibe

Terminal lottery wheel built with [Textual](https://github.com/Textualize/textual): press Space to spin the wheel and randomly pick a participant.

## Quick start

```cli
uvx rogvibe
# or
pip install rogvibe
rogvibe
# or
uv tool install --python 3.13 rogvibe
rogvibe
```

777 slot mode

```cli
uvx rogvibe --slot
```
